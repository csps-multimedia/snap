Liens externes

Objectif

Les liens externes sont tout simplement des liens qui conduisent à des sites Web en dehors du cours.

En cliquant sur ces liens, vous ouvrirez le site Web dans une nouvelle fenêtre.

Tous les liens doivent être indexés correctement dans la page Ressources pour fonctionner.

Pour accéder à la liste des liens d'indexation, ouvrez simplement les ressources dans la boîte à outils.

Exemple

Lorsque vous voulez trouver quelque chose, Google s'ouvrira dans une nouvelle fenêtre est l'endroit où aller; C'est donc un site Web externe.
