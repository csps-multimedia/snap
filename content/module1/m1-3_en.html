Policy on Government Security

It is important that individuals in all departments of the government understand their responsibilities in protecting personnel, safeguarding information and assets, and ensuring the continuity of operations. These requirements will be described in each of the modules throughout this course.

The Policy on Government Security stipulates how security should be managed within the federal government. This policy requires deputy heads to effectively manage security activities within their departments and to contribute to effective government-wide security management.

The policy is supported by the Directive on Security Management, which outlines requirements for a departmental security program. A number of mandatory procedures and standards identify more detailed security requirements within specialized areas of security known as security controls. These security controls are:

  • security screening
  • information technology security
  • physical security
  • business continuity management
  • information management security
  • security in contracts and other arrangements
  • security event management
  • security awareness and training
